# Altaroda Landing Page

Landing page para os projetos Altaroda, Garage 777 e Roda Suplente.

## 🚀 Deploy

Site disponível em: [Render](https://altaroda-landing.onrender.com)

## 📁 Estrutura

- `index.html` - Página principal
- `styles.css` - Estilos CSS
- `script.js` - JavaScript para animações
- `Assets/` - Logos e imagens

## 🎨 Design

- Dark mode com efeitos glassmorphism
- Totalmente responsivo
- Animações suaves de entrada
- Hover effects nos cartões

## 📦 Projetos

1. **Altaroda** - Equipamento, ferramentas e consumíveis
2. **Garage 777** - Car Care
3. **Roda Suplente** - Jantes e kits
